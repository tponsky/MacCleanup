## How to Use

1. **Click "Preview Cleanup"** to see all suggested actions
2. **Review each item** - check the description and version badges
3. **Select/deselect** actions using checkboxes
4. **Use "Select All"** if you trust all suggestions
5. **Click "Run Cleanup"** to execute only selected actions

## Example Workflow

```
1. Preview shows: "Presentation Deck: GCMD Marketing (1).pptx [DUPLICATE]"
   Original: "Presentation Deck: GCMD Marketing.pptx [LATEST]"
   
2. You see: Latest version is checked, duplicate is unchecked
   
3. You can: 
   - Keep both (uncheck delete on duplicate)
   - Delete duplicate (leave checked)
   - Review dates to confirm which is actually newer
```
