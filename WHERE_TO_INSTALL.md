## üåê For Others (Share It)

**Right now, the DMG only exists on your computer.** To share it with others, you need to upload it to:

### Option 1: GitHub Releases (Recommended)

1. Go to: https://github.com/tponsky/MacCleanup/releases
2. Click "Create a new release"
3. Tag: `v1.0`
4. Title: `MacCleanup v1.0`
5. **Upload** `MacCleanup-1.0.dmg` file
6. Click "Publish release"
7. **Share the link:** https://github.com/tponsky/MacCleanup/releases

**Then others can download from:**
- https://github.com/tponsky/MacCleanup/releases/latest
- Or: https://github.com/tponsky/MacCleanup/releases/latest/download/MacCleanup-1.0.dmg

### Option 2: AWS Website (If You Set It Up)

1. Deploy the website to your AWS EC2 server (see `website/deploy-to-ec2.sh`)
2. Upload `MacCleanup-1.0.dmg` to the website folder
3. Users can download from: `http://maccleanup.toddponskymd.com/MacCleanup-1.0.dmg`
